Student Instruments Product Page

A simple web page to display a list of musical instruments that can be filtered by category.


Features

Displays all instruments by default

Filter instruments by category: Woodwinds, Brass, Percussion, or All

Dynamic rendering of instrument cards using JavaScript

Responsive layout for different screen sizes


Technologies Used

HTML

CSS

JavaScript

How It Works

The instruments are stored in an array of objects in JavaScript.

When a category is selected from the dropdown, the instrumentCards function filters the array and generates HTML for each card.

The filtered cards are displayed dynamically in the .products-container div.
